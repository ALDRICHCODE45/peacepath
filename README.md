# Peace Path

psicologo virtual con capacidad de recordar conversaciones y crear meditaciones personalizadas

## Servidor de desarrollo

Instalacion

```bash
  git clone my-project
  cd my-project
  npm install
  docker-compose up -d
  bun dev
```
